# Compatibility

| Runtime | Modules | Status |
| --- | --- | --- |
| Node 22 | ESM | Supported |
| Node 24 | ESM | Supported |
| Node 26 | ESM | Supported |
| Node <22 | Any | Unsupported |
| Any Node | CommonJS | Unsupported |

The SDK is ESM-only. CI must exercise the oldest and newest available supported Node lines. Manifest schema version 1 targets Hoakbot 4.x.
